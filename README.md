# Tanjun

簡単な React Native の UI

## Installation

```sh
yarn add tanjun
```

## Usage

```jsx
import { View, Text } from 'tanjun';

// ... style will auto scale for your device correctly
const App = () => (
  <View style={{ width: 200, height: 140 }}>
    <Text style={{ fontSize: 46 }}>Hello Tanjun </Text>
  </View>
);
```
